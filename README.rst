############################
TDDBC for Python with Pytest
############################

これは、TDDBCのPythonista向け Pytest_ プロジェクトです

.. _Pytest: http://pytest.org/latest-ja/

動作確認環境
============

- Python 2.7.6
- Python 3.3.1
- PyPy 1.9.0
- PyPy 2.2.0

セットアップ
============

.. code-block:: sh

   $ pip install -r requirements.txt

**setup.py** を実行し

.. code-block:: sh

   $ python setup.py test
